# Peking-Railway-Volume-Visualization-Dashboard
# 北京市轨道交通客流智慧监测平台

## 项目简介

本项目是一个面向北京市轨道交通的客流数据可视化与分析工具，提供地图展示、多维度客流指标统计、站点筛选与对比、分时客流趋势分析等功能。平台基于 Web 技术构建，支持本地数据导入与实时交互，适用于交通规划人员、数据分析师及公众了解北京地铁客流特征。

## 主要功能

### 1. 客流地图展示
- 基于 Leaflet 的地图引擎，展示北京地铁站点分布。
- 站点以气泡形式呈现，气泡大小和颜色反映客流规模（可自定义颜色、透明度）。
- 支持底图切换（CartoDB 浅色底图）。
- 可选显示轨道交通线路骨架（GeoJSON 格式）。

### 2. 多维度筛选
- **线路级联筛选**：选择线路后，下方列出该线路所有站点，支持多选/全选/清空，地图联动显示选中站点。
- **环路级联筛选**：选择环路（如二环以内、三至四环等），列出该环路内站点，支持多选。
- **属性筛选**：按站点级别、区县、分区进行筛选。
- **名称搜索**：支持输入单个或多个站点（英文逗号分隔）精确匹配。
- **多边形选区**：在地图上绘制多边形，仅显示区域内站点。
- **客流阈值过滤**：仅显示全日进站量大于设定值的站点。

### 3. 客流指标看板
- **关键指标概览**：展示当前筛选条件下全日进站、出站、进出站总量。
- **分组对比图表**：按区县、环路、级别展示各分组的总量对比和单站均值对比（柱状图）。
- **单站全日进出站客流趋势**：上传分时 CSV 文件后，可查看任意站点一天内进站/出站客流曲线或柱状图，支持颜色自定义、图表类型切换。
- **历史客流对比**（对比模式）：选择基准年份和对比年份，地图上以色差气泡展示客流变化，并提供历年全网总量对比柱状图。

### 4. 数据导入与更新
- 支持通过界面按钮上传本地 CSV、GeoJSON 等文件，自动更新地图与图表。
- 默认读取同目录下的预设文件（如 `423站点基础属性信息.csv`、`2026年客流开发数据.csv`、`现状423座车站.json`、`420车站_站点单日进出客流量_20260311.csv` 等）。

## 文件依赖

请确保以下文件与 HTML 文件放置于同一目录下（或通过“导入本地数据”按钮上传）：

| 文件名 | 说明 |
|--------|------|
| `423站点基础属性信息.csv` | 站点基础属性（级别、区县、分区、环路） |
| `2026年客流开发数据.csv` | 年度客流数据（全日进站/出站量等） |
| `现状423座车站.json` | 站点地理坐标 GeoJSON 文件 |
| `420车站_站点单日进出客流量_20260311.csv` | 分时客流数据（含站点进站量/出站量表头） |
| `2025年底-线路909km-现状.json` | 线路几何数据（用于地图显示线路骨架） |

> 注：若缺少某些文件，页面顶部会显示红色提示，并可通过“导入本地数据”按钮手动上传。

## 使用方法

1. **打开网页**：双击 `轨道客流看板_级联筛选版.html` 文件，推荐使用 Chrome / Edge 浏览器。
2. **选择年份与客流指标**：顶部导航栏可选择年份（现状分析模式）或对比年份（历史对比模式），以及需要查看的客流指标（全日进站/出站/进出站量、早高峰等）。
3. **筛选站点**：
   - 在左侧“统计范围”面板中，使用线路/环路级联下拉框选择并勾选站点。
   - 使用“按级别/区县/分区筛选”按钮，弹出多选框进行属性筛选。
   - 在搜索框中输入站点名称（多站用英文逗号分隔）精确匹配。
   - 点击“多边形选区”按钮，在地图上绘制多边形，双击闭合完成区域筛选。
4. **调整地图样式**：右侧面板可修改气泡颜色、大小和透明度。
5. **查看客流趋势**：
   - 在“单站全日进出站客流趋势”板块，点击上传分时 CSV 文件（或自动加载默认文件），然后选择站点，即可查看分时折线/柱状图。
   - 在“各分组总量对比”和“各分组均值对比”中，可通过下拉框切换分组类型。
6. **历史对比模式**：点击顶部“历史客流对比”按钮，选择基准年份 A 和对比年份 B，地图气泡颜色将表示客流增减（红色增加、蓝色减少），并展示历年总量对比柱状图。

## 技术栈

- **前端框架**：原生 HTML5 / CSS3 / JavaScript (ES6+)
- **地图库**：Leaflet + OpenStreetMap 底图
- **数据处理**：PapaParse（CSV 解析）、Turf.js（地理空间分析）
- **图表库**：Chart.js（分组柱状图）、ECharts（分时客流趋势图）
- **数据格式**：CSV、GeoJSON

## 常见问题

**Q：为什么地图上没有显示任何站点？**  
A：请检查是否已导入或缺失必要的数据文件（尤其是 `现状423座车站.json`）。可通过“导入本地数据”按钮上传对应文件。

**Q：分时客流趋势图没有数据？**  
A：需要上传包含“站点进站量”和“站点出站量”双表头的 CSV 文件，或者确保默认文件 `420车站_站点单日进出客流量_20260311.csv` 存在于同目录。

**Q：线路级联筛选下拉框为空？**  
A：线路数据来源于车站 GeoJSON 中的 `线路` 属性，若文件中未包含，则无法自动提取。您可以手动在上传的 GeoJSON 中添加该字段，或通过“导入本地数据”补充。

**Q：历史对比模式下地图气泡颜色不变？**  
A：请确保已正确选择基准年份 A 和对比年份 B，并确认对应年份的客流数据文件已加载。
